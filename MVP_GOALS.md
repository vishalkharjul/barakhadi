# Barakhadi — MVP Goals

## Vision
A web app to help kids (~5 years old) learn tracing Marathi vowels in a fun way.

## Core Learning Features
1. **12 Marathi Vowels (स्वर) Tracing** — अ, आ, इ, ई, उ, ऊ, ए, ऐ, ओ, औ, अं, अः
2. **Guided Tracing** — Dotted/faded outlines showing where to draw
3. **Visual Feedback** — Simple celebration on completion (stars, checkmark animation)

## UX / Kid-Friendly Design
4. **Large, colorful UI** — Big touch targets, bright colors, minimal text
5. **Audio pronunciation** — Play the sound of each vowel when selected and on completion
6. **Simple navigation** — Icon-based, no reading required

## Technical Goals
7. **Mobile-first responsive web app** — Works well on tablets/phones
8. **Canvas-based tracing** — Smooth drawing experience
9. **No login/auth required** — Zero friction
10. **Lightweight & fast** — Minimal dependencies

## Out of Scope (for MVP)
- Consonants (व्यंजन), Barakhadi combinations, Numbers (1-10)
- Progress tracking / user accounts
- Gamification beyond basic feedback
- Handwriting accuracy scoring
- Offline/PWA support
